# Logo Instructions

Please save the 110 Barbershop logo image as "logo.png" in this directory.

The logo should be:
- File name: logo.png
- Location: assets/images/logo.png
- Recommended size: 200x200 pixels or larger (transparent background preferred)
- Format: PNG with transparency

The website is already configured to use this logo in:
- Navigation bar
- About section
- Footer

Once you add the logo image, the website will automatically display it.
